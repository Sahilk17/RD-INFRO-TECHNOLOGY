# Task 1 - Microcontroller Selection and Setup

## Objective
To select and configure an Arduino Uno microcontroller and upload a basic LED Blink program.

## Components Used
- Arduino Uno
- Wokwi Simulator

## Software
- Arduino IDE
- Wokwi Online Simulator

## Code

```cpp
void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);

  digitalWrite(13, LOW);
  delay(1000);
}
```

## Working
The Arduino Uno was configured successfully. A basic LED blink program was executed. The onboard LED blinked every one second.
